Turn 8 · User

Two bugs noticed:

  1. Double text appearing in the answer (every word duplicated).
  2. Code is not formatted properly when asking code-related questions.

Also update the KT file with the scheduler info, and add a VPS / AWS / DigitalOcean / GCP deployment guide to both KT and README.

Assistant

Bug 1 root cause: setTurns((prev) => { const arr = [...prev]; arr[i].content += piece; return arr; }) shallow-copies the array but mutates the same object. React StrictMode invokes the updater twice in dev to surface impurity → every token appended twice. Fixed by returning new objects everywhere:

const next = prev.slice();
next[idx] = { ...last, content: last.content + piece };
return next;

Bug 2 fix: strengthened ANSWER_SYSTEM in backend/lib/prompts.py — code MUST go in fenced blocks with explicit language tags; no inline raw code; no token repetition. Backend restarted to pick up the new prompt.

KT: added section 09 · Hourly Scheduler (env knobs, run modes, systemd snippet, single-process Qdrant warning) and section 10 · Deployment Guide (mermaid prod diagram; sizing table; 8 numbered steps: provision → bootstrap → app → Qdrant Docker → systemd API → frontend build → nginx + Let's Encrypt → ingest timer; provider commands for DO / AWS / GCP / Hetzner; ops notes).
